      Backup / Restore

      Backup tải file `finance.db`. Restore sẽ thay DB hiện tại và app tự tạo bản `.bak` trước khi ghi đè. Trên web public, phần này chỉ dành cho admin.

      Chưa có thao tác dữ liệu.

      PWA

      App có manifest và service worker để chạy giống app cài đặt trên máy hỗ trợ PWA.

      Đang kiểm tra PWA...

      Sửa lỗi phiên

      Dùng khi trình duyệt báo CSRF token không hợp lệ sau khi server restart, login/logout hoặc PWA cache bản cũ.
